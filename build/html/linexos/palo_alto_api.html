Pack PaloAlto by API Rest¶

Description¶

Checks pour l’API des équipements Palo Alto

Note

Ces checks passe par l’API Rest des Palo Alto si celle-ci est active. Ils fonctionnent avec un login/password associé à cette API. Le mode “read-only” est préférable pour ce compte.
