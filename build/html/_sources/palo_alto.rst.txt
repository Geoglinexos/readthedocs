===================
Pack PaloAlto
===================

***********
Description
***********

*Checks pour les équipements Palo Alto*

********
Services
********



Paloalto - App Version
        - Vérification de la version de l'application
        - SNMP

~~~~~

Paloalto - CPUs
        - Utilisation des CPU
        - SNMP

~~~~~

Paloalto - FANs
        - Vérification du fonctionnement des ventilateurs
        - SNMP

~~~~~~

Paloalto - Disks
        - Utilisations des disques / partitions
        - SNMP

~~~~~~

Paloalto - GlobalProtect gateway
        - Vérification de la passerelle GlobalProtect
        - SNMP

~~~~~~

Paloalto - HA
        - Statut d'un cluster High Availability (HA)
        - SNMP

~~~~~~

Paloalto - ICMP Sessions
        - Nombre de sessions ICMP
        - SNMP

~~~~~~

Paloalto - Panorama Status
        - Vérification du statut de Panorama 
        - SNMP

~~~~~

Paloalto - Stats $KEY$
        - Utilisation d'une ou plusieurs interfaces
        - SNMP

~~~~~~

Paloalto - System
        - Vérification de l'état du hardware
        - SNMP

~~~~~

Paloalto - TCP Sessions
        - Nombre de sessions TCP
        - SNMP

~~~~~~

Paloalto - Temperature
        - Vérification de la température
        - SNMP

~~~~~

Paloalto - Total Sessions
        - Nombre total de sessions liées au controller
        - SNMP

~~~~~~

Paloalto - UDP Sessions
        - Nombre de sessions UDP
        - SNMP

~~~~~

Paloalto - Version
        - Vérification de la version
        - SNMP
